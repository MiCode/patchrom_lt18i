Sampeyan dhedhemitan. Kaca sing dideleng ing jendela iki ora bakal katon ing riwayat peramban utawa riwayat telusur, lan ora bakal ninggal tapak tilas, contone cookie, ing piranti sawise sampeyan nutup jendhela dhedhemitan. Nanging file apa wae sing diundhuh utawa tandha sing digawe bakal disimpen.

Laku dhedhemitan ora mengaruhi tindak tanduke wong, server, utawa piranti lunak liyane. Ngati-ati karo:

  • Situs web sing nglumpukake utawa andum katrangan babagan sampeyan
  • Panyadhiya layanan internet utawa atasan sing nglacak kaca sing diparani
  • Piranti lunak jahat sing nglacak ketukan tombol kanggo ngijoli smiley gratis
  • Panitipriksa dening telik sandhi
  • Wong-wong sing ngadeg ing mburi sampeyan
