Sasaistīšana ar USB

Izmantojot USB kabeli, varat sasaistīt tālruni ar datoru, lai kopīgotu tālruņa interneta savienojumu ar datoru.
  • Sasaistīšanu var veikt operētājsistēmā Windows Vista, Windows 7 un Linux.

  • Tālruņa SD karti nevar likt uz datora, ja tas ir sasaistīts, izmantojot USB.

  • Plašāku informāciju, tostarp informāciju par sasaistīšanu ar USB citās operētājsistēmās, skatiet vietnē http://www.android.com/tether#usb.

Portatīvs Wi-Fi tīklājs

Tālruni var padarīt par portatīvu Wi-Fi tīklāju, lai kopīgotu tālruņa interneta savienojumu ar vienu vai vairākiem datoriem vai citām ierīcēm.
  • Ja tālrunis tiek lietots kā Wi-Fi tīklājs, nav iespējams izmantot tālruņa programmas, lai piekļūtu internetam, izmantojot tā Wi-Fi savienojumu.

  • Tīklāju var konfigurēt Wi-Fi tīklāja iestatījumos.

  • Plašāku informāciju skatiet vietnē http://www.android.com/tether#wifi.
