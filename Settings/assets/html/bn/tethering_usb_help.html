USB টেথরিং

আপনার ফোনের ইন্টারনেট সংযোগ কম্পিউটারের সাথে অংশীদারি করতে একটি USB কেবলের মাধ্যমে আপনি আপনার ফোনটিকে কম্পিউটারের সাথে আটকাতে পারেন
  • টেথরিং Windows Vista, Windows 7, এবং Linux-এর সাথে কাজ করে

  • USB আটকানো অবস্থায় আপনি আপনার ফোনের SD কার্ডকে আপনার কম্পিউটারে মাউন্ট করতে পারবেন না

  • অন্যান্য OS-এর সাথে USB টেথরিং ব্যবহার সহ আরও তথ্যের জন্য, http://www.android.com/tether#usb দেখুন
