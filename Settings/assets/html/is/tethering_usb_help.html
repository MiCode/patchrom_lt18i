USB-tjóðrun

Þú getur tjóðrað símann við tölvuna með USB-snúru, til að samnýta Internet símans með tölvunni
  • Tjóðrunin passar við Windows Vista, Windows 7 og Linux

  • Þú getur ekki fest SD-kort símans við tölvuna þegar USB er tjóðrað

  • Frekari upplýsingar, ásamt notkun USB-tjóðrunar með öðrum OS, sjá http://www.android.com/tether#usb
