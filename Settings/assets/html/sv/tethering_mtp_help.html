Internetdelning via USB

Du kan ansluta din telefon till datorn med en USB-kabel för att dela telefonens Internetanslutning med din dator
  • När du har installerat USB-drivrutinerna via PC Companion fungerar internetdelning med Windows Vista, Windows 7 och Linux

  • Du kan inte ansluta ditt telefonminneskort på datorn i läget Masslagring eller Medieöverföring

  • För mer information, inklusive hur du använder internetdelning via USB med andra operativsystem, gå till http://www.android.com/tether#usb

Bärbar Wi-Fi-hotspot

Du kan omvandla din telefon till en bärbar Wi-Fi-hotspot, för att dela din telefons Internet-anslutning med en eller flera datorer eller andra enheter
  • När din telefon fungerar som en Wi-Fi-hotspot kan du inte använda dina telefonprogram för att ansluta till Internet via dess Wi-Fi-anslutning

  • Du konfigurerar hotspot med Wi-Fi-hotspotinställningarna

  • För mer information, gå till http://www.android.com/tether#wifi
