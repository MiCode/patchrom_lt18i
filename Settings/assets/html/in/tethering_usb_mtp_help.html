USB tethering

Anda dapat tether telepon ke komputer menggunakan kabel USB, untuk berbagi sambungan Internet telepon dengan komputer
  • Tethering berfungsi dengan Windows Vista, Windows 7, dan Linux setelah Anda menginstal driver USB melalui PC companion

  • Anda tidak dapat menyambungkan kartu memori telepon pada komputer dalam Penyimpanan massal atau Mode Transfer Media

  • Untuk mendapatkan informasi selengkapnya termasuk cara menggunakan USB tethering pada SO lain, lihat http://www.android.com/tether#usb
