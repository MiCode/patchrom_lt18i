USB povezivanje

Možete povezati telefon i računar pomoću USB kabla, kako biste delili internet vezu telefona sa računarom.
  • Povezivanje funkcioniše sa operativnim sistemima Windows Vista, Windows 7 i Linux

  • Kada je USB povezivanje aktivno, ne možete stavljati SD karticu telefona na računar

  • Za više informacija, uključujući korišćenje USB povezivanja sa drugim OS, pogledajte http://www.android.com/tether#usb
