Uporaba USB kot modema

Svoj telefon lahko s kablom USB na računalnik priključite kot modem in tako internetno povezavo telefona delite z računalnikom.
  • Uporaba kot mode deluje z operacijskimi sistemi Windows Vista, Windows 7 in Linux

  • Kartice SD telefona ne morete namestiti v računalnik, ko se USB uporablja kot modem

  • Za več informacij, vključno z uporabo USB-ja kot modema z drugimi operacijskimi sistemi, glejte http://www.android.com/tether#usb
