USB-andmesideühendus

Kui ühendate oma telefoni arvutiga USB-kaabli abil, saate oma telefoni andmesideühendust arvutiga jagada
  • Andmesideühendus töötab operatsioonisüsteemidega Windows Vista, Windows 7 ja Linux

  • USB-andmesideühenduse kasutamise ajaks võite sisestada oma SD-kaardi arvutisse

  • Lisateavet, kaasa arvatud teavet USB-andmesideühenduse kasutamise kohta teiste operatsioonisüsteemidega, leiate http://www.android.com/tether#usb
