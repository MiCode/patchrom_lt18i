USB 網絡共享

您可使用 USB 傳輸線，將手機與電腦共享網絡，即讓您的電腦分享手機的互聯網連接
  • 網絡共享可在 Windows Vista、Windows 7 及 Linux 上使用

  • USB 網絡共享時，您無法將手機的 SD 卡掛接電腦

  • 欲知詳情，包括在其他操作系統使用 USB 網絡共享的資料，請瀏覽 http://www.android.com/tether#usb

流動 WLAN 熱點

您可將手機變成流動 WLAN 熱點，與一部或多電腦或其他裝置，分享手機的互聯網連接。
  • 手機用作 WLAN 熱點時，您無法使用手機的應用程式，經由 WLAN 連接存取互聯網

  • 使用 WLAN 熱點設定，設定熱點

  • 欲知詳情，請瀏覽 http://www.android.com/tether#wifi
