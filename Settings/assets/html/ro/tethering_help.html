Conectarea prin USB

Puteţi conecta telefonul la computer printr-un cablu USB pentru a partaja cu computerul conexiunea la Internet a telefonului
  • Conectarea funcţionează pentru Windows Vista, Windows 7 şi Linux

  • Nu puteţi instala cartela SD a telefonului pe computer când este conectat prin USB

  • Pentru informaţii suplimentare, inclusiv despre utilizarea conectării prin USB pentru alte sisteme de operare, consultaţi http://www.android.com/tether#usb

Punct cald Wi-Fi portabil

Puteţi transforma telefonul într-un punct cald Wi-Fi portabil pentru a partaja conexiunea la Internet a telefonului cu unul sau mai multe computere sau cu alte dispozitive
  • Când telefonul se utilizează ca punct cald Wi-Fi, nu puteţi utiliza aplicaţiile telefonului pentru a accesa Internetul prin conexiunea Wi-Fi a acestuia

  • Configuraţi punctul cald cu setările punctului cald Wi-Fi

  • Pentru informaţii suplimentare, vizitaţi http://www.android.com/tether#wifi
