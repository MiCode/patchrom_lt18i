Ngahubungkeun ku kabel USB

Anjeun bisa ngahubungkeun telepon Anjeun ka komputer Anjeun make kabel USB, pikeun ngabagikeun koneksi internet telepon Anjeun sareng komputer Anjeun
  • Ngahubungkeun ku kabel ieu bisa berpungsi sareng Windows Vista, Windows 7, jeung Linux

  • Anjeun teu bisa masangkeun kartu SD telepon Anjeun dina komputer Anjeun sawaktos kabel USB keur dihubungkeun

  • Kanggo inpo nu leuwih lengkep, kaasup nganggo cara ngahubungkeun ku kabel USB ieu sareng SO nu sejen, tingal http://www.android.com/tether#usb

Hotspot Wi-Fi Portabel

Anjeun bisa ngajadikeun telepon Anjeun jadi hotspot Wi-Fi portabel, pikeun ngabagikeun sambungan internet telepon Anjeun sareng hiji atawa leuwih komputer atawa alat nu sejenna
  • Sawaktos telepon Anjeun keur dijadikeun hotspot Wi-Fi, Anjeun teu bisa make aplikasi telepon Anjeun pikeun ngakses internet liwat koneksi Wi-Fi eta

  • Anjeun ngonpigurasikeun hotspot eta sareng setelan hotspot Wi-Fi

  • Kanggo inpo nu leuwih lengkep, buka http://www.android.com/tether#wifi
